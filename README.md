# weird-science
Repository dedicated to the versioning of TeX source files for unorthodox science stuff & papers-to-be.

## Papers in Development

### [Open-Ended Evolution in Multiagent Formal Systems: Curiosity, Novelty, and the Adjacent Possible](./open-ended-evolution-multiagent-systems/)
A study exploring open-ended evolution in multiagent formal systems, with a focus on curiosity, novelty, and the adjacent possible.

### [Complex Systems as Unbounded Dynamical Holarchies: A Framework for Understanding and Simulating Emergent Phenomena](./complex-systems-dynamical-holarchies/)
A framework for understanding and simulating emergent phenomena in complex systems through the lens of unbounded dynamical holarchies.

## Repository Structure

Each paper has its own dedicated folder with the following structure:
- `src/` - LaTeX source files
- `figures/` - Figures, diagrams, and illustrations
- `references/` - Bibliography files and reference materials  
- `drafts/` - Draft versions and working documents

## Contributing

This repository uses comprehensive LaTeX .gitignore rules to keep auxiliary files out of version control while preserving source files and important assets.
