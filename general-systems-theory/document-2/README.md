# Open-Ended Evolution in Multiagent Formal Systems

Exploring curiosity, novelty, and the adjacent possible in multiagent formal systems.

## Overview

This document explores open-ended evolution in multiagent formal systems, with a focus on mechanisms of curiosity, novelty discovery, and the adjacent possible.

## Structure

- `main.tex` - Main LaTeX document
- `sections/` - Document sections:
  - `introduction.tex` - Introduction to the topic
  - `methodology.tex` - Research methodology
  - `results.tex` - Findings and results
  - `conclusion.tex` - Concluding remarks
- `references/` - Bibliography files

## Building

To build this document locally, run:
```bash
pdflatex main.tex
bibtex main
pdflatex main.tex
pdflatex main.tex
```

Or use the automated GitHub Actions workflow which builds PDFs on every push.
